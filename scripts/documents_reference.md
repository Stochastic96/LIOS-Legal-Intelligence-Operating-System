### Chunking strategy for 1M questions
- Article-level chunks (current): ~1 chunk/article → ~10 questions/chunk = 87,000 questions
- **Sub-article 400-word overlapping windows**: ~8 windows/article → ~2.5 questions/window = ~1,000,000 questions
- Use `lios/ingestion/document_indexer.py` after uploading PDFs via mobile UploadScreen or SSH

### Upload method
1. **Mobile app** → Upload tab → select PDF → uploads to `/api/upload` → indexed instantly
2. **SSH to VM** → `python3 scripts/download_docs.py` → bulk download + index
3. **Direct curl**: `curl -X POST http://VM_IP:8000/api/upload -F "file=@doc.pdf" -F "regulation=CSRD"`
